.. _shapes:

Формы
-----

Здесь перечислены все доступные формы. Каждая форма имеет своё представление в рендере.
Все они наследуются от базовой формы, наследуя её параметры и поля.

Параметры
=========

Это аргументы, которые можно задать в шаблоне, они влияют на то как форма будет рендериться.

Пример: создание формы  с заданием параметров.

.. code-block:: json

    {
        "type": "shape-type", "x": 0, "y": 0,
        "parameter": "value"
    }

Поля
====

Это атрибуты формы, доступные в шаблоне для получения определённых данных.

Многие поля являются параметрами, которые заданы при создании формы, но есть
и те, которые рассчитываются автоматически и доступны в шаблоне.

Например, указав позицию и размер формы, мы получаем поле ``left``, возвращающее самую левую
точку формы в глобальных координатах.

Пример: вторая форма ссылается на поле ``left`` первой формы. Это позволяет создать зависимость между положением форм.

.. code-block:: json

    {
        "shapes":[
            {
                "type": "rect", "x":0, "y": 100, "id": "shape1",
                "width": 100, "height": 100
            },
            {
                "type": "rect", "x": 0, "y": "shape1.left",
                "width": 100, "height": 100
            }
        ]
    }

.. note:: Все указанные параметры доступны в шаблоне под теми же именами

Далее в документации будут указаны доступные **параметры** и **поля** для каждой формы.

.. note:: Форма наследует все параметры и поля родительской формы.
          В документации будут указаны только новые для этого класса параметры и поля
