Frame Stamp
-----------

Пакет для добавления информации на кадры используя шаблон и контекст с переменными.

Установка
=========

Добавление в проект с помощью `poetry`

.. code-block:: bash

   poetry add https://github.com/paulwinex/frame-stamp.git

Локальная разработка и запуск

.. code-block:: bash

   git clone https://github.com/paulwinex/frame-stamp.git
   cd frame-stamp
   make install

Запуск диалога разработки шаблона

.. code-block:: bash

   make run

Сборка документации

.. code-block:: bash

   make docs
